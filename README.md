# mRremoteNgPasswordDecryptor

This project is designed to retrieve information about saved connections from mRemoteNG. It uses the mRemoteNG configuration file to extract the necessary data, such as the connection path and master password (if specified).

## Запуск

1. Navigate to the folder with the executable file (`PasswordDecrypor.exe`) using the command line:
   ```bash
   cd path\to\dir\with\exe
  ```bash
  PasswordDecrypor.exe -p [опциональный путь к XML файлу] -master [опциональный мастер-пароль]
